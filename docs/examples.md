# Examples

This document contains practical examples.

## Basic Usage

Here's a basic example of how to use the documentation:

1. Connect to the MCP server
2. List available resources
3. Read the resource you need

## Advanced Usage

For more complex scenarios, you can:

- Query multiple documents
- Cross-reference between documents
- Build workflows based on documentation content

## Tips

- Always check the available resources first
- Use the resource templates to understand the URI format
- Reference related documentation using the `docs://` URI scheme
